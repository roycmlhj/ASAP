# frontend

## Project setup
```
npm install
```

### Vue Bootstrap

```
npm install vue bootstrap bootstrap-vue
```

### Bootstrap

```
npm install bootstrap@4.5.3
```

### Axios

```
npm install axios
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
